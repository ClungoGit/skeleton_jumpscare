# Skeleton Jumpscare

A Silksong Mod based of the Minecraft mod "Running Skeleton Jumpscare"

https://www.curseforge.com/minecraft/mc-mods/running-skeleton-jumpscare

Causes a random skeleton event on average once every 250 seconds (based off the Minecraft mod).
You can change this in the BepInEx config file.
You can also manually trigger a skeleton event by pressing F6.
